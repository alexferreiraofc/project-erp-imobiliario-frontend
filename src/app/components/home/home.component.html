ERP Imobiliario Sparky


Olá, seja bem vindo ao meu sistema de ERP para imobiliárias. O objetivo principal desse projeto é criar um sistema que visa utilizar técnicas modernas para desenvolver um software focado no setor imobiliário, por ser um setor complexo muitos sistemas pecam em fornecer soluções flexiveis e amigáveis ao usuário. Um exemplo é que ainda é muito comum treinamento para usar esses sistemas, e alguns deles ainda são soluções desktop.

Inicialmente o projeto será construido apenas com o módulo de Vendas, será possível acompanhar o progresso da negociação e após finalizado a venda é disparado uma mensagem via API do Telegram para o gestor ou grupo de gestores notificando a nova venda, dessa forma acompanhando em tempo real a finalização da venda e eliminando a necessidade do gestor verificar negociação por negociação. É uma dor que identifiquei na empresa que trabalho atualmente e após a construção colocarei a aplicação em produção para uso dessa empresa. Será mantido no github uma cópia do projeto principal como base.

Tecnologias utilizadas:
- Angular 15
- Jquery
- Spring Boot 3
- Postgres 15
- Docker
- Ngx-Bootstrap
- Telegram API
